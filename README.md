# Todo List App

A React application for managing a list of todos.

## Installation Instructions

1.  Clone the repository:
    ```
    git clone https://github.com/Olena-Salnikova/olena-salnikova-react-todo-list
    ```
2.  Navigate to the project directory:
    ```
    cd olena-salnikova-react-todo-list
    ```
3.  Install dependencies:
    ```
    npm install
    ```

## Running the Development Server

1.  Start the development server with:
    ```
    npm run dev
    ```
2.  Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.
